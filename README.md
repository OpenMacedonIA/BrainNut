**The Brain of WatermelonD.**

BrainNut provides the advanced reasoning and AI capabilities for the WatermelonD system. Like a walnut, it has a hard shell (robust architecture) protecting complex logic and memory on the inside.

## Features
- **Gemma 2B Integration**: Local LLM for natural conversation.
- **MANGO T5**: Specialized component for translating natural language into bash commands.
- **Memory Management**: Handles long-term memory and context.

## Integration
This module is intended to be used as a submodule within [WatermelonD](https://github.com/OpenMacedonIA/WatermelonD).
